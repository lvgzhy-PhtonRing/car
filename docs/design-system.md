# 记账管理系统 v2.0 — 视觉设计规范

> 借鉴艺术/创意类网站的视觉语言，以**画廊式克制美学**重构应用的色彩与图标体系。

---

## 1. 设计理念

| 原则 | 说明 |
|------|------|
| **画廊式留白** | 减少视觉噪音，用负空间引导注意力 |
| **克制用色** | 低饱和主色 + 高饱和强调色 = 1:9 法则 |
| **温度一致** | 所有颜色保持暖基调，不混入冷白/冰蓝 |
| **图标线描化** | 统一 1.5px 描边 SVG 图标，替代 emoji |

## 2. 色彩系统

### 2.1 中性色 — Warm Neutral（基座）

| Token | 色值 | 用法 | 角色 |
|-------|------|------|------|

## 3. 图标系统

### 3.1 图标来源

使用 **Lucide** (lucide.dev) 开源 SVG 图标库，统一 1.5px 描边、24×24 视口。

> Lucide 是 Feather Icons 的社区维护版，约 1500+ 图标，MIT 协议。

CDN 引用：
```html
<script src="https://unpkg.com/lucide@latest"></script>
```
用 `data-lucide` 属性注入，或编译为内联 SVG。

## 4. 关键组件样式更新

### 4.1 玻璃面板 (Glass)
保留现有 `.glass` 系列，仅将底色从纯白微调为暖白：
```css
.glass {
    background: rgba(245, 240, 234, 0.62);  /* #f5f0ea base */
    /* 其余不变 */
}
```

## 5. 实现路径

### Phase 1 — CSS 变量化（推荐）
1. 在 `:root` 中定义所有色值 CSS 变量
2. 逐步替换内联颜色引用

### Phase 2 — 图标替换
1. 引入 Lucide CDN 或编译为内联 SVG
2. 逐个替换 emoji 为 Lucide 图标
3. 确保 `data-lucide` 自动替换或手动渲染

### Phase 3 — 微调
1. 替换背景渐变
2. 微调玻璃面板透明度
3. 调整间距和留白
