🗄️ localStorage 数据结构设计

单键 carAppData 存储完整应用状态，兼容 JSON 导入导出

📦 顶层结构
{
  "version": "2.0.0",
  "accounts": { "pocket": {...}, "invest": {...} },
  "cars": [ Car, Car, ... ],
  "transferPairs": [ TransferPair, ... ]
}
💰 Account（账户）
{
  "id": "pocket",           // "pocket" | "invest"
  "name": "零花钱",
  "monthlyAllowance": 500,     // 仅 pocket; invest 为 null
  "allowanceDay": 1,          // 每月几号; 仅 pocket
  "transactions": [ Transaction, ... ],
  "loans": [ Loan, ... ]           // 仅 invest; pocket 为 []
}
📝 Transaction（交易记录）
{
  "id": "txn_1706000000_abc123",
  "type": "expense",          // "income" | "expense"

                              // loan_new | loan_repay
                              // transfer_in | transfer_out | other
  "amount": 120.00,
  "date": "2026-06-15",
  "carSid": "a1b2c3d4-...",   // null if not car-related

  "note": "闲鱼购入"
}

🔄 TransferPair（转账配对）
{
  "id": "xf_1706000000",
  "fromAccount": "invest",    // 转出账户
  "toAccount": "pocket",      // 转入账户
  "amount": 1000.00,
  "date": "2026-06-20",
  "note": "投资盈利转回零花钱"
}
💡 每次转账在双方账户各生成一条 Transaction（type 相反），共享同一个 transferPairId。TransferPair 作为独立记录保存转账详情。

📊 余额计算规则（实时计算，不存储）

指标 计算方式
账户余额 sum(income) - sum(expense) 该账户所有交易
零花钱余量 pocket 账户余额
投资款余额 invest 账户余额
已投资小车总额 sum(buyPrice) 所有 category=invest 且未售出的小车
库存小车预期价值 sum(expectedValue) 所有未售出小车
近三月零花钱消费 pocket 账户按月份 groupBy → sum(expense)
